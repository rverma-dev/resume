# Technical Director Platform Sample

This folder is a reference for Technical Director / Head of Platform resume
tailoring.

- `resume.md` shows the content augmentation strategy for build-a-function
  Director, Head of Platform, and AI Infrastructure leadership roles.
- `resume.pdf` is the rendered sample. It should follow the clean A4 layout in
  `resume/references/director-clean-layout-reference.pdf`: no decorative border,
  compact two-page output, no clipped text, and no download control in the PDF.
- Render application PDFs by iterating browser print zoom in the 65-75% range.
- Keep experience in reverse chronology; do not fold recognizable brands such
  as Atlassian, Sprinklr, or Rokitt into a generic earlier-experience bucket.
- For Director roles, lead with the team and operating-scope stories that are
  defensible: Brane 2->40, Niki 3 EMs / 40 engineers, and Salesforce de-facto
  platform-program leadership. Do not imply formal Salesforce line management.

Use this sample as the default fallback for Director applications when a freshly
generated PDF is not at least as clean as the reference layout.
