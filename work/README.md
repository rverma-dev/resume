# Work Evidence

Store factual evidence for resume tailoring here. Suggested folders:

```text
ADRs/
RFCs/
design_docs/
planning/
architecture/
incident_reviews/
project_docs/
collaboration/
```

Generated resumes may only use claims backed by `resume/base_resume.md` or
documents in this tree.

Private company-internal source maps belong under `work/private/`. That path is
ignored by git because the public resume site must not publish internal repo,
Slack, or document references.
